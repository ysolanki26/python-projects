# Password Strength Analyzer (Python)

## Project Overview
This project is a Python-based Password Strength Analyzer that evaluates the strength of a password using multiple security criteria such as length, character types, and pattern detection. It generates a detailed report including strength, score, risk level, crack time, and suggestions.

## Objectives
- To analyze password strength using multiple factors  
- To improve user awareness about password security  
- To simulate real-world password validation systems  

## Features
- Password strength detection (Weak, Medium, Strong)  
- Score system (0–100)  
- Risk level analysis (High, Medium, Low)  
- Estimated crack time  
- Suggestions for improving password  
- Pattern detection (123, abc, repeated characters)  
- Common password detection  

## Technologies Used
- Python  
- Regular Expressions (re module)  

## How to Run
1. Open the project folder in VS Code  
2. Open terminal  
3. Run the command:
   python app.py  
4. Enter a password when prompted
